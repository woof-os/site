Woof os: note on installation

Home · Qtile Sway · Pacman Repository · Important Note

Sometimes, during installation you may encounter this error:

The source file system "/run/archiso/bootmnt/arch/x86_64/airoootfs.sfs" does not exist

This just means that the USB/CD you booted form isn't mounted to bootmnt.

To proceed with the installation:

  1. Run lsblk to list all your block devices.
  2. Find the partition of your USB/CD that contains the Woof OS live image.
  3. Mount the device to /run/archiso/bootmnt.
  4. Run calamares to start the installer and proceed with the installation as usual.

If any issues persist, contact us on Discord or send a message from the footer, and tell us how we can contact you.
